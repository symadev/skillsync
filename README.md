## âš™ï¸ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/symadev/skillsync.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd skillsync
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

4. **Start development server**
   ```bash

   ```

   ```bash
   yarn dev
   ```
